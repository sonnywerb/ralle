RALL-E
Robot Activity Loop for Live Engagement

## installation
1. download latest version of the installer .exe from Releases.
2. after downloading, run the installer as admin.
3. ralle.exe will be installed in C:/Program Files/ralle.

## usage
- run ralle.exe to start the program
- there will be a system tray icon present if it's running
- to stop the program: right click on the icon > Stop RALLE

RALL-E
Robot Activity Loop for Live Engagement

## installation
1. download latest version of the installer .exe from Releases.
2. after downloading, run the installer
3. select whether you want to create start menu/desktop shortcut
4. RALL-E.exe will be installed in C:\Users\<user>\AppData\Local\RALL-E

## usage
- run RALL-E.exe to start the program
- there will be a system tray icon present if it's running
- to stop the program: right click on the icon > Stop RALL-E
